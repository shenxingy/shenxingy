# VERIFY

Checkpoints for the GitHub profile README. All links, HTML structure, and content accuracy must pass before pushing.

## Checkpoints

| # | Area | Scenario | Status |
|---|------|----------|--------|
| 1 | HTML | All HTML tags are properly opened and closed | ✅ |
| 2 | Links | All `<a href>` URLs are reachable (2xx/3xx) | ✅ |
| 3 | Email | mailto address matches contact info | ✅ |
| 4 | Content | No broken/empty `<a>` tags (href not empty, text not blank) | ✅ |
| 5 | Length | README stays within 200–300 lines per CLAUDE.md guidelines | ✅ |
